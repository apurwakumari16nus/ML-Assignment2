## Project summary

End-to-end NLP pipeline analysing emotional shifts on Reddit during the Feb–Apr 2026 Iran-Israel-US conflict, compared against a truly peaceful pre-war baseline (Apr–Jun 2025). Five classifiers (VADER, TF-IDF+LogReg, TF-IDF+SVM, fine-tuned BERT, fine-tuned DistilBERT) are evaluated and cross-validated against an independent 7-class HuggingFace model with a neutral class, plus a non-parametric bootstrap 95% CI on the deflated shift.

**Headline finding (one sentence):** Our fine-tuned 6-class BERT reports a +4.2pp war-vs-baseline distress rise (p < 1e-5), but cross-validation against an independent 7-class neutral-aware model on the same 2,000+2,000 sample deflates this to +0.85pp with a 95% bootstrap CI of [−2.00, +3.80] — directionally positive but statistically fragile, **not** a mental-health crisis.

## Reproducibility statement

Per the assignment requirement ("all results stated in your report must be reproducible from the repository"), **every number, table, and figure in the report can be regenerated by running the scripts in this repo in numerical order**. Two caveats:

1. **Saved model weights are not in this repo** (4 GB + 252 MB, over GitHub's 100 MB file limit). They are regenerated by running `05_bert_train.py` and `07_distilbert_train.py` (combined ~1 hour on Apple Silicon MPS). Training is seeded (`seed=42/123/456`) and deterministic up to driver-level float nondeterminism — expect F1 to match the report to within ±0.005.
2. **Live Reddit fetches** (`01_reddit_fetch_war.py`, `02_reddit_fetch_baseline.py`) depend on the public Reddit JSON API. Posts can be deleted over time, so a fresh run may return slightly fewer comments than the 96,879 war-period / 5,240 baseline counts in the report. **To bypass this**, the cleaned CSVs from the original fetch are shipped in the repo — reviewers can skip 01–03 entirely.

## End-to-end reproduction (the path reviewers should take)

Since the cleaned CSVs are shipped, you can skip the slow Reddit fetch and training-evaluation goes straight to modelling:

```bash
# 1. Setup
pip install -r requirements.txt

# 2. Stage B — training (required; regenerates the two model folders)
python 04_vader_sentiment.py           # seconds, no training needed
python 05_bert_train.py                # ~10 min on MPS — creates bert_emotion_model/final/
python 07_distilbert_train.py          # ~45 min on MPS — creates distilbert_emotion_model/final/

# 3. Stage C — classify Reddit and generate analyses
python 08_bert_classify.py             # ~15 min
python 09_distilbert_classify.py       # ~10 min
python 10_tfidf_logreg.py              # seconds
python 11_tfidf_svm.py                 # seconds
python 12_temporal_analysis.py
python 13_subreddit_analysis.py
python 14_vader_bert_comparison.py
python 15_error_analysis.py
python 16_model_comparison.py          # generates Figure 1 + Table 6
python 17_event_significance_test.py   # Bonferroni over 8 events

# 4. Stage D — cross-validation (the A+ stage)
python 18_baseline_comparison.py       # +4.2pp headline
python 20_check_pretrained_vs_finetuned.py  # HF vs our BERT on 1k sample
python 21_check_hf_baseline_vs_war.py       # HF deflates headline to +0.85pp
python 22_bootstrap_hf_shift.py             # 95% CI on the deflated shift
```

**Optional scripts (not needed for the core results):**
- `01_reddit_fetch_war.py`, `02_reddit_fetch_baseline.py`, `03_clean_data.py` — only if you want to re-fetch from Reddit; the cleaned CSVs are already shipped.
- `06_bert_train_rigorous.py` — 3-seed × 3-LR rigor study for the report's Table 6 (~90 min); not required for downstream scripts.
- `19_reddit_manual_label.py` — optional domain-validation script (exports 200 comments for external labelling).

**Total reproduction time:** ~2.5 hours on Apple Silicon MPS, ~5 hours on CPU.
